# About
KAP-INATOR is a suite of tools that process tournament data in real time.

# Tools
## Mat Velocity Predictor
### Input
- JSESSIONID: A cookie captured from a browser page logged into arena. Inspect -> Application -> Cookies -> JSESSIONID.
- Event ID: The ID of the tournament. Copied from the web url of arena dashboard.
- Start Time: The time which the first match began. HH:MM AM/PM format.
- Total Bouts: The total bouts for your event
